Scripts

Many of these scripts require Erica Utilities from Cydia

Simply click the link at the beginning of each description to install
Have any questions? Contact me!

appnames - pulls all of your apps and provides the bundle ID (helpful for themers) and provides exact file location for App Store apps if run with "loc"
ex. ./appnames.sh loc

appstatus - originally built as part of another project. Pulls the same information as appnames, but only for stock iOS and Cydia apps. This script will also tell you if the app is stock or from Cydia.
Warning: Takes a very long time to execute

redeb - put debain packages back together! Just run the script followed by a package ID (package IDs can be found at the bottom of the page in Cydia)
ex. ./redeb.sh com.ipadkid.package

teamid - intended for iOS 10 to easily pull their team id to be used with Cydia Extender Installer. Requires Yalu102 to be installed with the Apple ID you intend to use. Can be used directly with cyextender using the pipe function
ex. ./teamid.sh | cyextender

tweaklist - simple tweak list dump. Can be used with "id" or "name" to dump package IDs (devs may ask for this) or for tweak names (users may ask for this)
ex. ./tweaklist.sh id
