FTT

FTT is short for Flex to Theos. This bash script does its best to do exactly that, turn Flex patches into full Theos projects

Known issues:

  • Patches labeled as systemwide by Flex may not work properly
  • init methods (has a + prefix) may not work properly

Latest version is available here: Version 5

  • Added color handling
  • Support for multiple variables in non-void methods
  • Proper support for only converting one patch

Older versions: Version 4

  • Full void handling
  • Handling for id mostly handled
  • Way faster, IA

Older versions: Version 3
