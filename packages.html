About the packages on my Cydia repo

DevRepos adds repos that have a lot more, and some newer, terminal commands
DevCommands will then install a bunch of helpful commands

OpenSSH for Yalu adds the SSH utilities to devices that use Dropbear
Enable Dropbear over WiFi allows devices to be SSHed into over WiFi (Yalu102 beta 5 and newer are locked to only over USB tunneling by default)

EW39.iWidget is a highly modified version of Evelyn's EW39.iWidget.
The track name and artist are shown above the dock. Currently only optimized for iPhone 6 resolution.

Live Photos Enabler allows the iPhone 6 and 6 Plus to take Live Photos

NotoLines allows you to choose how long notification messages are (lockscreen and Notification Center)

NtSpeed is a fork of Julio's that adds a preference panel to move the data indicator box

SwitchServices for Notifications enables iOSCreatix's SwitchService tweak for iOS 10 notifications

AlwaysGrabber makes the grabber to show up on notifications when 3D Touch is enabled (iOS 10)
Response to this Reddit request

No Vol Slider gets rid of any volume sliders loaded from Springboard. I believe this is just the lockscreen slider and the one in control center, but I'm not entirely sure

Disable Control Center does just that

Local Cydia Repo creates a local Cydia repository for a user to put whatever Debian packages they'd like in, for ease of use
